# API Documentation

## Category API

### Get All Category IDs

**Request:**
```
GET /categories
```
**Parameters:**
- `page` (optional, default: `${default.page}`)
- `size` (optional, default: `${default.size}`)
- `includeUrls` (optional, default: `false`)

**Response:**
```json
[
  1,
  2,
  3
]
```

### Get Category by ID

**Request:**
```
GET /categories/{id}
```
**Path Parameters:**
- `id` (required)

**Response:**
```json
{
  "categoryId": 1,
  "group": "Group Name",
  "name": "Category Name",
  "description": "Category Description"
}
```
